# Soccer Analysis

## Installation

```bash
git clone https://github.com/gabrielfonseca/soccer-analysis.git
cd soccer-analysis
pip install -r requirements.txt
```

## Usage

### Backend

```bash
cd backend
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
